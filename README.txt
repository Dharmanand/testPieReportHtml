If you want to use loder.js file from ur github account use following urls
 1. Use this URL in production: https://cdn.rawgit.com/Dharmanand/testPieReportHtml/efbffc75/resources/charts/google/loader.js
 2. Use this URL for development: https://rawgit.com/Dharmanand/testPieReportHtml/master/resources/charts/google/loader.js
 ***************************************************************************************
 If want to create your own resource access url go to "https://rawgit.com/" and paste ur github file path.
 Example:
 Git Url: https://github.com/Dharmanand/testPieReportHtml/blob/master/report/index.html
 Use this URL in production: https://cdn.rawgit.com/Dharmanand/testPieReportHtml/8e3f3da6/report/index.html
 Use this URL for development: https://rawgit.com/Dharmanand/testPieReportHtml/master/report/index.html
 ***************************************************************************************
 You can put git file url this like: https://raw.github.com/[username]/[repository]/[branch]/[filename].js
 Example:
 Git url of js file: https://github.com/Dharmanand/testPieReportHtml/blob/master/resources/charts/google/loader.js
 Open url of js file: https://raw.github.com/Dharmanand/testPieReportHtml/master/resources/charts/google/loader.js
 Limitation: Only file contents will open. If it'll be html page the html source codes will open.
